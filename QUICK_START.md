# 快速开始指南

## 🚀 立即编译测试

### 1. 打开项目
- 双击 `CGDrawingSystem.slnx` 在 VS2026 中打开

### 2. 选择配置
- 顶部工具栏：选择 **Debug | x64** 或 **Release | x64**

### 3. 编译
- 按 **F7** 或菜单：**生成 → 生成解决方案**
- 等待编译完成

### 4. 运行
- 按 **F5** 或菜单：**调试 → 开始调试**
- 或直接运行生成的 exe：
  - Debug: `x64\Debug\CGDrawingSystem.exe`
  - Release: `x64\Release\CGDrawingSystem.exe`

## 🔧 如果编译失败

### 常见错误1：找不到头文件
**错误信息**：`无法打开包含文件: "xxx.h"`
**解决**：
1. 检查文件是否存在
2. 检查项目文件中的路径是否正确
3. 重新添加文件到项目

### 常见错误2：链接错误
**错误信息**：`无法解析的外部符号`
**解决**：
1. 检查所有 `.cpp` 文件是否在项目中
2. 检查函数声明和定义是否匹配
3. 清理后重新生成

### 常见错误3：资源编译错误
**错误信息**：`Menu.rc` 相关错误
**解决**：
1. 检查 `Menu.rc` 是否在项目中
2. 检查 `resource.h` 是否存在
3. 检查资源 ID 是否冲突
